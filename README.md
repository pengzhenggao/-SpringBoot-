
# power
vue后台管理系统模板
见文章: https://juejin.im/post/5e670edde51d4527110a9b40

# 说明
>默认(主)版本为基础框架版

丰富组件 请访问 2.0版本: https://github.com/cgq001/admin-menu/tree/2.0


## 初始化
```
yarn install
```
### 运行
```
yarn run serve
```
### 打包
```
yarn run build
```
